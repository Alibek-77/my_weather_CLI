# My Weather CLI Application

A lightweight, efficient Command Line Interface (CLI) application built with Python to fetch and display real-time weather information for any city using a Weather API.

This project showcases clean Python architecture, structured data handling with **Pydantic**, environment variable safety, and proper dependency management.

---

## 🚀 Features

- **Real-time Weather Data:** Fetches up-to-date weather metrics (temperature, condition, humidity, wind speed).
- **Robust Validation:** Uses `Pydantic` to parse and validate JSON responses safely.
- **Secure Configuration:** Protects API keys using `python-dotenv`.
- **Developer-Friendly CLI:** Clear output formatting directly in your terminal.

---

## 🛠️ Tech Stack & Concepts Covered

- **Language:** Python 3.10+
- **Data Validation:** Pydantic (BaseModel)
- **HTTP Client:** Requests
- **Environment Management:** python-dotenv, venv, requirements.txt
